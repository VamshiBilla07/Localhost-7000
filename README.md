# NexBank - Premium Fintech Portal

A sophisticated, localized, and high-performance banking web application built with React.js and Vite.

## 🚀 Features
- **Send Money**: Instant transfers with recipient selection and real-time summaries.
- **Pay Bill**: Manage utility payments (Electricity, Water, Internet) with digital receipts.
- **Add Card**: Issue virtual and physical cards with live 3D previews and spent limit controls.
- **Localized UI**: Specialized for the Indian market with **₹ (INR)** currency and `en-IN` formatting.
- **Advanced Analytics**: Interactive spending overview charts using Chart.js.
- **Security First**: Simulated secure processing and transaction confirmation states.

## 🛠️ Tech Stack
- **Frontend**: React.js (Vite)
- **Routing**: React Router
- **Charts**: Chart.js & react-chartjs-2
- **Styling**: Vanilla CSS (Glassmorphism & Monochrome theme)

## 🏁 Getting Started
1. Clone the repository
2. Run `npm install`
3. Start the dev server with `npm run dev`
